## About

BhojanBridge is a platform designed to facilitate food sharing between individuals and organizations. It connects people who have surplus food with those in need, creating a sustainable and compassionate community solution to food scarcity and waste.

The name "BhojanBridge" combines the Sanskrit word "Bhojan" (meaning food) with "Bridge" to symbolize our mission of creating bridges between surplus food and those who need it most.

## ✨ Features

- **User Registration & Authentication**: Secure account creation and login system
- **Food Listing**: Donors can post available food items with details
- **Search & Discovery**: Recipients can easily find available food offerings
- **Donation Tracking**: Monitor the status of food donations
- **Community Engagement**: Connect with local food donors and recipients
- **Responsive Design**: Mobile-friendly interface for accessibility
- **Real-time Updates**: Stay informed about new listings and availability

## 🚀 Installation

### Prerequisites

- Web server (Apache/Nginx)
- PHP 7.0 or higher
- Python 3.6 or higher
- MySQL/MariaDB database

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aditya-Roy-code/BhojanBridge.git
   cd BhojanBridge
   ```

2. **Set up the database**
   - Import the database schema (if available)
   - Configure your database credentials

3. **Configure PHP settings**
   - Update database connection in `php/database.php`
   - Set appropriate file permissions

4. **Start the web server**
   ```bash
   # For local development
   php -S localhost:8000
   ```

5. **Access the application**
   - Open your browser and navigate to `http://localhost:8000`

## 💻 Usage

### For Donors
1. Create an account or log in
2. Click "Post Food" to list surplus food items
3. Provide details: food type, quantity, expiry date, location
4. Wait for interested recipients to reach out

### For Recipients
1. Sign up and create a profile
2. Browse available food listings
3. Contact donors for collection/delivery details
4. Arrange pickup or delivery

### Admin
- Monitor platform activity
- Manage user accounts and listings
- Track donation statistics
